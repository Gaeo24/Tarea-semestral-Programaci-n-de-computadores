# Tarea-semestral-Programación-de-computadores
¿Quién gana en este juego? El o la jugadora que logre destruir el
 tanque enemigo más veces en 300 turnos o quien destruya el tanque enemigo 3 veces, lo que ocurra
 más rápido.
 
Para probar el codigo los archivos (Tarea2_main.c, map.c, map.h) deben estar en la misma carpeta. 
Compilar el programa con el comando gcc -o juego.exe Tarea2_main.c map.c

Al iniciar el juego se pueden realizar 3 acciones:
-movimiento con (w/a/s/d)
-disparo con (u/h/j/k)
-terminar el programa al ingresar 'e'
